# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.2.0] - 2026-05-04

### Changed
- Default benchmark from VFINX to SPY
- Upgrade pvbt dependency to v0.9.2

## [0.1.0] - 2026-05-02

### Added
- Initial release of Vigilant Asset Allocation strategy
- Recency-weighted 13612W momentum signal across offensive and defensive sleeves
- Generalized breadth-momentum allocation supporting both VAA-G4 (aggressive) and VAA-G12 (balanced) variants via the `BreadthThreshold` parameter
